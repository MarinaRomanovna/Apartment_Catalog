# Apartment_Catalog

This is simple CRUD application for adding, editting, reading, deleting apartment objects.
App is under development.

Technology stack:

-MVVM

-ViewBinding

-RXJava3

-Dagger 2

-Room

-Kotlin Flows

-Cicerone

-Material Design.
